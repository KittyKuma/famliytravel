## 여는 법
- **가장 쉬운 방법:** `index.html` 더블클릭 → 바로 열립니다(빌드 불필요).
- **폰에 설치(PWA)·오프라인까지 쓰려면** 웹서버로 열어야 합니다:
  ```
  cd D:\_MyProject\Osaka
  python -m http.server 8099
  ```
  브라우저에서 `http://localhost:8099` 접속 → (모바일) 메뉴 → **홈 화면에 추가**.
- **인터넷에 올리려면** 폴더 통째로 GitHub Pages / Netlify / Vercel에 올리면 끝(정적 사이트).

## 폰에서 쓰는 법 (가족에게 공유)
1. 위 방법으로 사이트를 폰 브라우저에서 연다.
2. 공유 → **홈 화면에 추가** → 앱처럼 아이콘 생성.
3. 한 번 열어두면 **오프라인(비행기/데이터 없이)에서도** 일정·회화·지도링크 사용 가능.
